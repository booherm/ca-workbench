CA Workbench
----------------------------------------
ESC          - Quit
PRINT SCREEN - Take a screenshot
PAUSE        - Toggle iteration pause
G            - Toggle grid lines
W            - Toggle auto iterate
ENTER        - Iterate 1 step

One Dimensional CA
----------------------------------------
LEFT ARROW   - Decrease rule set number
RIGHT ARROW  - Increase rule set number
DOWN ARROW   - Decrease rule number
UP ARROW     - Increase rule number


Random Boolean Network
----------------------------------------
SPACE        - Reset RBN
R            - Refresh input sites
F            - Feed output back to feedback input
T            - Toggle automatic feed output back to feedback input on each iteration
A            - Toggle automatic new input on each iteration
V            - Toggle site connections
I            - Toggle fading of inactive sites
UP ARROW     - Increase RBN connectivity by 1
DOWN ARROW   - Decrease RBN connectivity by 1
LEFT ARROW   - Turn off neighborhood connections
RIGHT ARROW  - Turn on neighborhood connections
KEYPAD 7     - Increase external input rows by 1
KEYPAD 1     - Decrease external input rows by 1
KEYPAD 8     - Increase feedback input rows by 1
KEYPAD 2     - Decrease feedback input rows by 1
KEYPAD 9     - Increase external output rows by 1
KEYPAD 3     - Decrease external output rows by 1
INSERT       - Shift input data
